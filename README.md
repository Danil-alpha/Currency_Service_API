# Currency Service API

Асинхронный сервис для парсинга, хранения и визуализации курсов валют на основе данных ЦБ РФ.

### Что умеет:
1. Автоматически скачивает актуальные курсы топ-8 валют (USD, EUR, CNY и др.) с API ЦБ РФ.
2. Кэширует данные в PostgreSQL. Повторные записи на одну дату игнорируются (идемпотентность).
3. Генерирует PNG-графики трендов за указанный период (работает в памяти через `io.BytesIO`, файлы на диск не пишутся).
4. Обновляет базу автоматически один раз в день в 12:00 МСК (время публикации курсов ЦБ) через FastAPI Lifespan. Расчет времени до следующего запуска динамический, таймер не сдвигается при перезапуске контейнера.

## Стек
* Python 3.11 / FastAPI (Async)
* PostgreSQL 16
* SQLAlchemy 2.0 / Alembic (миграции)
* Docker / Docker Compose
* HTTPX, Matplotlib

---

## Как запустить

### 1. Переменные окружения
Создайте файл `.env` в корне проекта по примеру из `.env.example`:
```bash
cp .env.example .env
```

### 2. Сборка и запуск в Docker
Проект полностью контейнеризирован. База данных, автоматическое применение миграций Alembic и само приложение поднимаются одной командой:

```bash
docker compose up --build
```

* API будет доступно по адресу: `http://localhost:8000`
* Интерактивная документация (Swagger): `http://localhost:8000/docs`

---

## API Эндпоинты

* `GET /` — Проверка доступности сервиса (Health-check).
* `POST /currencies/refresh` — Ручное принудительное обновление курсов из ЦБ за текущий день.
* `GET /currencies/chart?code=USD&days=30` — Получение PNG-графика динамики курса. Параметр `code` — код валюты, `days` — глубина истории.

## Структура проекта
* `app/main.py` — Точка входа, фоновый планировщик (Lifespan).
* `app/routes.py` — Эндпоинты FastAPI.
* `app/services.py` — Логика работы с ЦБ и построение графиков.
* `app/models.py` — Описание таблиц для SQLAlchemy.
* `alembic/` — Папка с миграциями базы данных.
* `docker-compose.yaml` & `Dockerfile` — Конфигурация контейнеров (для сборки Python-приложения используется зеркало для стабильности скачивания зависимостей).
